# Total Recap Live

Explore more than a decade of MTV countdown history from 1998 through 2008.

The site includes searchable songs and artists, year and metric filters, multi-series timelines, daily countdowns, saved playlists, CSV exports, and Apple Music previews.

## Local development

```bash
npm install
npm run dev
```

## GitHub Pages

Push to `main`. The included GitHub Actions workflow builds the site and deploys it to GitHub Pages.
